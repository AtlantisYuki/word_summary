# 省流总结助手（Chrome 插件）

支持两种总结方式：

- 选中网页中的一段文字，点击「总结选中文本」
- 不选中文本，点击「总结整页内容」
- 快捷键触发：默认 `Alt+S`（选中总结）、`Ctrl+Alt+S`（整页总结）
- 右键菜单触发：支持「总结选中文本」「总结当前网页」「识图总结」

交互增强：

- 选中总结后，原文会加高亮底色，并在末尾生成「省流」气泡，点击气泡弹出摘要框
- 图片识图总结后，图片会高亮并生成「省流」气泡，点击气泡弹出摘要框
- 全文总结会显示右侧悬浮总结窗，支持收起/展开与关闭
- 高亮底色可在设置页修改
- Popup 页面仅保留「标题 / 设置 / 更多」
- 更多页面支持文字、图片、文字+图片混合总结（支持粘贴图片和上传图片）

支持 OpenAI API 兼容供应商配置：

- 多供应商（可新增/删除/启停）
- 每个供应商可分别配置 `textModel` 与 `visionModel`
- 每个供应商可配置反应时长 `timeoutMs` 与失败重试次数 `maxRetries`
- 每个供应商可分别配置文字优先级 `textPriority` 与视觉优先级 `visionPriority`
- 支持开发调试日志（可查看供应商调用、重试、超时、错误详情）
- 视觉总结内置多种 JSON 结构自动降级（提升不同供应商的兼容性）
- 对 200 非 JSON 响应支持流式/文本解析（提升兼容代理网关能力）
- 扩展重载后自动恢复失效页面上下文（自动重注入 content script）
- 摘要小窗支持一键复制
- 选中文本快捷键
- 整页总结快捷键
- 高亮底色

## 文件结构

- `manifest.json`：扩展配置（Manifest V3）
- `content.js`：提取选中文本/整页文本
- `background.js`：调用 OpenAI 兼容接口进行总结
- `popup.html` `popup.css` `popup.js`：弹窗 UI 与交互
- `more.html` `more.css` `more.js`：更多总结页面（文字/图片/混合）
- `options.html` `options.js`：配置页面

## 本地使用

1. 打开 Chrome，进入 `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择当前目录
4. 打开扩展的「选项」，配置一个或多个供应商（Base URL、Key、文字模型、视觉模型、优先级、超时、失败重试）
5. 在任意网页点击扩展图标，打开「更多」可输入文字/图片进行总结
6. 可在设置页修改快捷键和高亮颜色，保存后立即生效

## 接口兼容说明

插件请求地址固定为：

- `Base URL + /v1/chat/completions`（若 Base URL 已包含 `/v1` 或完整路径会自动兼容）

请求体采用 OpenAI Chat Completions 风格（`model` + `messages`）。
