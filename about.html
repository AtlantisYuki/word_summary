关于省流总结助手

一个支持文本、网页、图片多场景总结的浏览器插件，兼容 OpenAI API 风格服务商。

当前版本 v-
打开最新发布页
点击“检查更新”可对比 GitHub 最新版本（仅提示，不会自动升级）。

主要能力

  • 选区总结、全文总结、识图总结
  • 多供应商配置与优先级回退
  • 可配置快捷键、加载文案、选区样式
  • 支持调试日志，便于排查接口问题

隐私与数据

插件仅在你触发总结时调用你配置的 API。配置和日志存储在浏览器本地存储中，不会自动上传到其他服务器。
